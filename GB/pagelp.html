LINGUAGENS WEB

São linguagens de programação projetadas para desenvolver aplicativos, páginas e conteúdo específico para a web. Elas são usadas para criar sites interativos, dinâmicos e funcionais, permitindo que os desenvolvedores criem uma ampla variedade de recursos, desde simples páginas da web até aplicativos complexos baseados em navegador.

OQUE USAMOS

    HTML
    CSS
    JAVASCRIPT
    PHP
    RUBY
    TYPESCRIPT

UTILIZAÇÃO NA PRÁTICA

    HTML (HYPERTEXT MARKUP LANGUAGE):
    Usado para definir a estrutura e o conteúdo básico de uma página web. Marca elementos como títulos, parágrafos, listas, imagens, links e formulários. Define a hierarquia e organização dos elementos na página.
    CSS (CASCADING STYLE SHEET):
    Usado para controlar o layout, estilo e aparência visual de uma página web. Define cores, fontes, margens, espaçamentos, posicionamento e animações. Ajuda a garantir a consistência de design em todo o site.
    JAVASCRIPT:
    Usado para adicionar interatividade e funcionalidades dinâmicas a páginas web. Manipula elementos da página em resposta a ações do usuário. Pode ser usado para criar animações, validações de formulário, interações de usuário em tempo real e muito mais.
    PHP
    Usado principalmente no lado do servidor para criar páginas web dinâmicas. Gera conteúdo de página com base em dados de bancos de dados ou outras fontes. Interage com bancos de dados, processa formulários e executa lógica do lado do servidor
    RUBY
    Usado com o framework Ruby on Rails para criar aplicativos web eficientes. Faz o manuseio do lado do servidor, gerenciamento de banco de dados e outras operações complexas.
    TYPESCRIPT
    Usado para adicionar tipagem estática ao JavaScript e tornar o desenvolvimento mais seguro. Compatível com o ecossistema JavaScript e usado principalmente no lado do cliente.
